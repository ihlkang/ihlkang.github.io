远程服务器搭建SVN

远程服务器上安装SVN

安装

  • 查看svn版本:rpm -qa subversion
  • 安装svn：yum install subversion
  • 创建新的目录:mkdir svn
  • 在目录下新建svn仓库,注意(cd到svn目录下新建):svnadmin create test

配置

  • 配置用户名及密码:vim passwd

  • 配置新用户的授权文件:vim authz

启动svn版本库:

  • 启动

    svnserve -d -r /home/centos/test

  • 关闭远程服务器防火墙

    service iptables stop
